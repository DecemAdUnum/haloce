Welcome to Halo Custom Edition!

What is Halo Custom Edition?

Halo Custom Edition is a game that serves as a free extension of Halo: Combat Evolved that allows players to play custom-made maps. Modders can create their own scenery, vehicles, weapons, multiplayer maps and single player maps for the Halo Custom Edition game.

How do I download Halo Custom Edition?

In order to install Halo Custom Edition, you need to own Halo: Combat Evolved for the PC. Using the CD Key you got with Halo: Combat Evolved, Followthis guideto install the Halo Custom Edition game.
